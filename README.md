# Анализ пользовательских отзывов на Яндекс Картах


**Цель:** исследовать связь между текстом отзыва и оценкой заведения на платформе Яндекс Карты. Выявить ключевые слова и темы, характерные для позитивных и негативных отзывов.

**Источник данных:** [Geo Reviews Dataset 2023](https://huggingface.co/datasets/d0rj/geo-reviews-dataset-2023) — открытый датасет отзывов об организациях в России (январь–июль 2023).

**Объём данных:** 500 000 отзывов.

**Инструменты:** Python, pandas, scikit-learn, NLTK, pymorphy2, matplotlib, seaborn.


## Бизнес-вопрос (гипотетический)

*Платформа агрегатор отзывов хочет автоматически определять главные проблемы пользователей, чтобы помочь владельцам заведений быстрее реагировать на негатив.*

**Главный вопрос исследования:** какие слова и темы наиболее характерны для негативных и позитивных отзывов?

## Структура анализа


1. Загрузка и первичный анализ (EDA)
2. Оценка дисбаланса и распределений
3. Очистка и лемматизация текстов
4. Частотный анализ ключевых слов
5. TF-IDF для уникальных маркеров тональности
6. Семантическая кластеризация слов
7. Формулировка выводов и гипотез


## Результаты

### 1. Распределение оценок

| Оценка | Количество | Доля |
|--------|------------|------|
| 5 | 390 515 | 78.1% |
| 4 | 41 160 | 8.2% |
| 3 | 21 686 | 4.3% |
| 2 | 12 088 | 2.4% |
| 1 | 34 351 | 6.9% |
| 0 | 200 | 0.0% |

**Вывод:** наблюдается сильный дисбаланс — почти 80% отзывов имеют максимальную оценку. Это важное ограничение: датасет не отражает равномерно все мнения.

### 2. Длина отзыва как маркер тональности

| Тональность | Средняя длина (слова) |
|-------------|----------------------|
| Негативная | 46.0 |
| Нейтральная | 37.7 |
| Позитивная | 24.7 |

**Вывод:** негативные отзывы в среднем в **два раза длиннее** позитивных. Пользователи тратят больше слов, чтобы объяснить причины недовольства.

### 3. Топ-20 слов по тональности

| Позитивные отзывы | Негативные отзывы |
|-------------------|-------------------|
| хороший (128k) | сказать (13k) |
| место (96k) | просто (11k) |
| отличный (85k) | день (11k) |
| персонал (85k) | человек (9.8k) |
| спасибо (72k) | время (9.7k) |
| магазин (66k) | место (9.1k) |
| рекомендовать (65k) | заказ (8.5k) |
| вкусный (64k) | работать (7.8k) |

**Вывод:** позитив связан с эмоционально окрашенными словами («отличный», «спасибо», «вкусный»). Негатив содержит больше глаголов и существительных, описывающих конкретные проблемы.

### 4. Уникальные маркеры (TF-IDF)

Слова, которые **специфичны** для каждой тональности:

| Позитивные (уникальные) | Негативные (уникальные) |
|-------------------------|-------------------------|
| отличный | сказать |
| спасибо | деньги |
| вежливый | ужасный |
| вкусный | заказ |
| уютный | ждать |
| чисто | отвратительный |

**Вывод:** уникальные маркеры негатива — это либо сильные негативные эмоции («ужасный», «отвратительный»), либо конкретные действия/объекты («заказ», «ждать», «деньги»).

### 5. Семантические кластеры (негативные отзывы)

Методом кластеризации KMeans (на основе совместной встречаемости слов) выделены следующие темы:

| Кластер | Размер | Ключевые слова | Тема |
|---------|--------|----------------|------|
| Кластер 1 | 46 слов | день, время, заказ, цена, деньги, ждать, минута, час | **Время и деньги** |
| Кластер 2 | 250 слов | персонал, ужасный, отвратительный, еда, отношение, плохой | **Качество сервиса** |

**Вывод:** в негативных отзывах выделяются **две основные проблемные зоны** — качество обслуживания/продукта (больший кластер) и время ожидания с ценой (меньший кластер).



## Гипотезы для дальнейших исследований

На основе полученных результатов можно сформулировать следующие гипотезы (требуют проверки на дополнительных данных):

1. **Время ожидания — основной драйвер негатива**  
   *Проверить:* сравнить частоту слов «ждать»/«очередь» в отзывах с низкими и высокими оценками внутри одной категории заведений.

2. **Пользователи редко пишут длинные позитивные отзывы**  
   *Проверить:* проанализировать корреляцию между длиной отзыва и оценкой после фильтрации ботов/накруток.

3. **Качество персонала влияет на оценку сильнее, чем цена**  
   *Проверить:* сравнить кластер «персонал» и «цена» по их предсказательной силе (например, через логистическую регрессию).
